# 📚 React Library App

A responsive React application for browsing and exploring books.  
Built as part of my React learning path and extended with clean UI, reusable components, and real e‑commerce‑style features.

This project demonstrates component architecture, routing, state management, and dynamic UI behavior — all core skills for front‑end development.

---

## 🚀 Features

- **Dynamic Routing** using React Router  
- **Book Detail Pages** with URL parameters  
- **Cart System**  
  - Add to cart  
  - Update quantity  
  - Price calculation helpers  
- **Sorting & Filtering**  
  - Price: Low → High  
  - Price: High → Low  
  - Rating  
- **Skeleton Loading States** for images  
- **Reusable Components** (Rating, Price, Book, Nav, Footer)  
- **Clean State Management** using React hooks  
- **Helper Functions** for price logic and totals  
- **Responsive Layout** for mobile, tablet, and desktop  

---

## 🧠 What I Learned

- Structuring a React project with reusable components  
- Managing state through props and hooks  
- Handling dynamic routes with `useParams`  
- Implementing loading states with image preloading  
- Debugging common React issues (NaN values, missing params, function references vs calls)  
- Sorting and filtering data using helper functions  
- Building UI patterns similar to real e‑commerce apps
